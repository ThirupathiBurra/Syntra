An AI-powered Enterprise Operating System designed to automate complex, multi-step workflows using autonomous agents, while maintaining strict human-in-the-loop security.

## ❓ Problem Statement

In today's fast-paced corporate environment, businesses face two critical bottlenecks that slow down productivity and scale:

1. **Autonomous Workflow Agents:** Employees spend thousands of hours manually executing multi-step administrative workflows that span across different departments and databases. Existing automation tools are rigid, rule-based, and cannot dynamically adapt to changing requirements or reason through complex problems.
2. **AI Knowledge Assistant:** Teams struggle with fragmented knowledge. When employees or AI models need to retrieve accurate internal company policies or data, they often face "hallucinations" or spend hours searching through unorganized documents.

---

## 💡 Solution

**Syntra OS** is an intelligent platform that acts as a "Mission Control" for your enterprise. It solves these problems by deploying **Agentic Workflows**:
- Instead of rigid flowcharts, Syntra uses a multi-agent orchestration engine (LangGraph) that can dynamically plan, reason, and execute tasks based purely on a single natural language command. 
- It features an advanced **Knowledge Base** powered by RAG (Retrieval-Augmented Generation), allowing agents to search and read specific enterprise documents instantly, ensuring every action they take is grounded in your company's actual data.
- Most importantly, it introduces **Human-in-the-Loop (HITL)** security. High-risk actions generated by the AI automatically pause and request human authorization before execution.

---

## ✨ Features

- **Multi-Agent Swarm Orchestration:** Planners, Reviewers, and Executors dynamically collaborate to break down and solve complex tasks.
- **Enterprise Knowledge Base (RAG):** Upload PDFs, Markdown, and TXT files. Syntra automatically chunks, embeds, and stores them in a Vector Database for instant semantic retrieval.
- **Human-in-the-Loop Approvals:** A dedicated dashboard for managers to review AI-proposed payloads and explicitly authorize or reject them.
- **Real-Time Execution Graph:** Watch the AI "think". The platform streams the agent's thought process and execution tree live to the UI via Server-Sent Events (SSE).
- **Executive Analytics Dashboard:** Monitor the ROI of your AI workforce, tracking hours saved, automation volume, and workflow success rates.

## 🚀 Installation

Follow these steps to set up Syntra OS on your local machine.

### Prerequisites
Make sure you have installed:
- Node.js (v20+)
- Python (v3.11+)
- [uv](https://github.com/astral-sh/uv) (Fast Python package manager)

### 1. Clone the repository
```bash
git clone https://github.com/ThirupathiBurra/syntra-ai-workflow-platform.git
cd syntra-ai-workflow-platform
```

### 2. Backend Setup
```bash
cd backend
cp .env.example .env
# Add your Gemini, Groq, and Supabase keys to the .env file

uv sync
uv run uvicorn app.main:app --reload --port 8000
```

### 3. Frontend Setup
```bash
cd ../frontend
cp .env.example .env.local
# Set NEXT_PUBLIC_API_URL=http://localhost:8000/api/v1

npm install
npm run dev
```

## 📖 Usage

1. **Launch the App:** Open `http://localhost:3000` in your browser.
2. **Train the Brain:** Navigate to the **Knowledge Base** and upload your enterprise documents (e.g., Financial Reports, HR Handbooks).
3. **Execute Workflows:** Go to the **Workflow Studio** and type a complex request in natural language (e.g., "Analyze our Q3 finances based on the latest report and draft a summary email").
4. **Monitor:** Watch the **Active Workflows** execution graph to see the agents planning and reasoning in real-time.
5. **Authorize:** If an agent attempts a high-risk action, navigate to the **Approvals** tab to review and approve the task.
